Description
==

Hi There,

I'm a simple plugin which adds a few columns to your posts dashboard.

 - Word Count
 - Except Count
 - Image Count
 
I'll color them according to a set limit in the settings menu.

Allowing you at a glance to see which posts you need to improve.
